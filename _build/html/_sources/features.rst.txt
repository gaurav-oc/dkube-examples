Features
############

Following are the list of features implemented in the release **R0.1**.


*   Multi user portal

*   GitHub OAUTH integration

*   DL workflows

*   Versioning of data and models

*   Integration with TensorFlow framework

*   Seamless integration with GPU fabric

*   Experiment on laptop and move training to DC cluster

*   Serving model on DC cluster

*   Logs & ELK stack integration

*   ksonnet based release package (no upgrade)

*   Eval keys, docker hub

*   Manual testing
